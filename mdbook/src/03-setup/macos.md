# macOS

All the tools can be installed using [Homebrew]:

[Homebrew]: http://brew.sh/

``` console
$ # ARM GCC debugger
$ brew install arm-none-eabi-gdb

$ # Minicom
$ brew install minicom

$ # lsusb lists USB ports
$ brew install lsusb
```
